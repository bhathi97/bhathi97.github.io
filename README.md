# Portfolio Website

This is a static portfolio website built with plain HTML, CSS, and JavaScript, ready for GitHub Pages deployment.

## Files

- `index.html` contains the page structure and editable content.
- `styles.css` contains the full design and responsive layout.
- `script.js` adds mobile navigation and reveal animations.

## Customize

1. Open `index.html`.
2. Replace the placeholder name, summary, experience, projects, and contact links with your real CV details.
3. Add your images inside the `images/` folder.
4. Update the image paths in `index.html` if your filenames are different.
5. If you want, add your real resume PDF to this folder and link it from the page.

## Add Images

- Profile photo: replace `images/profile.jpg`
- Project images: replace `images/project-1.jpg`, `images/project-2.jpg`, `images/project-3.jpg`
- Gallery images: replace `images/gallery-1.jpg` to `images/gallery-4.jpg`

Use JPG, PNG, or WebP files. Keeping images around 1200px wide or less will help the site load faster on GitHub Pages.

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Push to the `main` branch.
4. In GitHub, open `Settings` > `Pages`.
5. Set the source to `Deploy from a branch`.
6. Choose the `main` branch and `/ (root)` folder.
7. Save, then wait for GitHub to publish the site.

Your portfolio will be available at:

`https://your-github-username.github.io/repository-name/`
